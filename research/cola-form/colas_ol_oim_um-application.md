Follow these steps to upload a label image to the application:

1. From the Create Application Step 3 of 3: Upload Labels page, select the **add/remove Images** button. The Upload Label Images page displays. See Figure 75 and Figure 76.
2. Select the **Browse** button to select an attachment to upload.

**Note:** Each image file must:

- Have a file type of **JPG** or **TIFF** (file extensions: **.jpg/.jpeg/.jpe** or **.tif/.tiff**)
- Not exceed **750KB** in size
- Have a compression/quality ratio set at **Medium** (7 out of 10 or 70 out of 100)
- Utilize colors in the **RGB** color mode, **not** colors in the **CMYK** color mode
- Have **no** surrounding white space or printer's proof detail; this must be cropped out

**Note:** See **How to Prepare Images for Upload** for more information.

**Note:** TIFF files should **not** be saved with JPG compression.

3. Select the attachment type from the drop-down list; i.e., tell us which label this is (brand, neck, back, etc.).
